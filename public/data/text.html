Ook kan het andersom, je constateert een string, maar je wilt er een nummer van maken.

Number.parseInt()

Hiervoor heb je de Number.parseInt() methode voor nodig. deze verwacht 2 parameters van je.

  1. De eerste parameter string, zorgt ervoor dat de string veranderd in een nummer
  2. De tweede parameter radix, wordt gebruikt in de conversie. als er een hoeveelheid van 10 is, word het [0, 9].

Je kan deze twee parameters dan ook niet overslaan, anders weet het systeem niet hoe die moet tellen.
